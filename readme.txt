0.[vertifypic文件夹和.idea文件夹不要删]

1.安装依赖库文件，在编译器的terminal终端输入下面的代码+回车
pip3 install -r requirements.txt

2.修改config文件里面的netid账号、场地、时间、邮箱

3.直接在前一天晚上，运行main程序，有空场地会自动预定并发送通知

4.（选用）在运行main之前设置电脑自动关机，在shutdown里填入希望关机的时间。打开cmd命令提示符，输入下面语句）： 
shutdown /s /t （shutdown运行结果）

  如果要取消自动关机计划，在命令提示符输入：
  shutdown /a
